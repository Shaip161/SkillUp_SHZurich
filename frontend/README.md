# Frontend Workspace

This area covers the user-facing online experience for TransitionAI Zurich.

## Product flow

- onboarding and profile capture
- CV upload and profile review
- transition recommendations
- job-detail and skill-gap exploration
- learning roadmap and agent-assisted guidance

## Structure

- `src/app/`: route-level screens and flow containers
- `src/components/`: reusable UI components
- `src/features/`: product features grouped by domain
- `src/lib/`: API clients, utilities, and shared helpers
- `src/styles/`: theme tokens and global styles
- `public/`: static assets
